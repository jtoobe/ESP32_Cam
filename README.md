Para cargar un programa en la ESP32CAM con el modulo USB lo que se debe hacer es lo siguiente:
1) Pulsar el boton IO0 ANTES de conectar el cable USB
2) cargar el programa desde la IDE de Arduino
3) Pulsar el boton RST o desconectar y conectar para que se ejecute el programa
